### kakaoTalk Chat bot Simple Example
- node.js 카카오 챗봇을 공부하려고 만든 간단한 예제입니다
- 근처 구내식당 메뉴를 알려줍니다( 내용은 수기로 수정. 단순한 응답기능만 있음 )
- 월,화,수,목,금 이라는 메시지를 보내면 요일에 해당하는 구내식당 메뉴를 대답합니다

### 실행방법
- install node.js
- node app.js
